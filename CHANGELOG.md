# Changelog

## 1.0.2 — 2026-06-11

### Исправления

- Автообновление работает при установке в папках с кириллицей и другими не-ASCII символами в пути (PowerShell вместо batch-файла)

## 1.0.1 — 2026-06-11

### Исправления

- Корректное обнаружение и удаление моделей Ollama (манифесты + blobs)
- Корректное обнаружение и удаление моделей Hugging Face Hub (models--* + общий blobs)
- Поддержка Ollama в Docker-томах и нестандартных путях
- Исключение дублирования родительских каталогов (Ollama, Hugging Face, ComfyUI)
- Исправлен запуск framework-dependent сборки (отключён InvariantGlobalization)

### Улучшения интерфейса

- Реальные имена моделей Ollama из манифестов
- Обрезка длинных путей и названий с подсказками при наведении
- Кнопка «Открыть папку» в колонке пути
- Сортировка по клику на заголовок колонки
- Сохранение ширины колонок между запусками

### Сборка и дистрибуция

- Два варианта поставки: portable (self-contained) и framework-dependent (~4 МБ)
- Framework-dependent требует [.NET 8 Desktop Runtime](https://aka.ms/dotnet/8.0/windowsdesktopruntime-x64)
